# miniquery


## Usage/Examples

```javascript
// Adding class
  MQ(selector).AC('class');

  // Removing class
  MQ(selector).RC('class');

  // Toggling class
  MQ(selector).TC('class');

  // Adding click event
  MQ(selector).CLICK(function() {
      MQ(this).AC('class');
  });
```
